## Core points

- Fincept Terminal v4 is presented as a native C++20 + Qt6 financial desktop terminal with embedded Python analytics.
- It combines market data connectors, quant modules, trading flows, and AI-agent style strategy assistance.
- The project uses dual licensing (AGPL + commercial), signaling both open and enterprise positioning.

## Why it matters

It adds a concrete example of AI-enabled domain tooling outside core agent framework research.
