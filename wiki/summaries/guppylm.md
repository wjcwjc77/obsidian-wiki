# GuppyLM Tiny Educational Language Model

- Topics: [[wiki/topics/small-models-and-tools]]

## Core points

- GuppyLM is a tiny transformer project designed to make LLM training easier to understand.
- It can be trained on modest hardware and is small enough to run in the browser.
- The project is explicitly educational rather than a production agent system.

## Why it matters

It gives the corpus a lightweight model-building reference point outside the agent-systems cluster.
