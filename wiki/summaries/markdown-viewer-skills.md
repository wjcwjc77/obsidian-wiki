## Core points

- This repository packages visualization-oriented skills for AI coding agents directly in Markdown workflows.
- It covers multiple diagram ecosystems (Mermaid, Vega, Graphviz, UML, and related formats).
- It includes installation and skill packaging conventions to standardize usage.

## Why it matters

It enriches the skills corpus with documentation/visualization capabilities, not only coding or business API tasks.
