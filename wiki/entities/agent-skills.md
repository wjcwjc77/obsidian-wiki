# Agent Skills

## Role in this corpus

Agent skills is the broad bucket for reusable domain packs, operating knowledge, and task-specific capability bundles that agents can apply across repeated workflows. In this corpus it follows `raw/sources/agent/skills/` and emphasizes how concrete domains are turned into stable agent-facing knowledge modules.
