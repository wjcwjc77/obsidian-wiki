# Agent RAG

## Role in this corpus

Agent RAG is the broad retrieval bucket for systems that treat retrieval as an explicit part of agent reasoning rather than a hidden similarity lookup. In this corpus it follows `raw/sources/agent/rag/` and includes structure-aware, vectorless, and multimodal retrieval workflows.
