# Agent Memory

## Role in this corpus

Agent memory is the broad bucket for how agents persist, consolidate, retrieve, and structure context across longer-running workflows. In this corpus it follows `raw/sources/agent/memory/` and also connects to systems that make memory inspectable and reusable in practice.
